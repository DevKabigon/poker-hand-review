# Design Fix 04 - Floating Nav + Landing Redesign

작성일: 2026-02-28  
범위: Phase 1 / Step 4 (랜딩 플로팅 네브 + 본문 폭 정렬 + 랜딩 대폭 리디자인)

## 목표

- 랜딩 상단바를 플로팅 네브로 전환
- 네브 폭을 본문 컨텐츠 폭(`DefaultContainer`)과 일치
- 랜딩 전체 구조를 더 모던하고 밀도 높은 형태로 재설계
- 과도한 스크롤 길이 완화

## 변경 파일

- `src/app/[locale]/(marketing)/page.tsx`
- `src/components/brand/brand-logo.tsx`
- `src/components/ui/button.tsx`

## 적용 내용

1. 플로팅 네브 도입 + 폭 정렬
- 기존 `sticky` 헤더를 `fixed` 플로팅 네브로 변경
- 네브를 `DefaultContainer` 내부에 배치해서 본문 폭과 정확히 동일하게 정렬
- 반투명 카드형 배경 + blur + soft shadow로 상단 레이어 명확화
- 네브 구성:
  - 로고
  - 섹션 앵커(`features`, `how-it-works`)
  - 대시보드 이동, 새 핸드 시작, 언어/테마 토글

2. 히어로 섹션 재설계
- 1열 중심에서 `2열(카피 + 플로우 카드)` 구조로 변경
- 핵심 배지/헤드라인/설명/CTA를 한 블록으로 재배치
- 오른쪽에 Step 요약 카드 배치해 제품 동작 흐름을 즉시 인지 가능하게 구성
- 플로팅 네브와 겹치지 않도록 상단 패딩 재조정

3. 랜딩 섹션 구조 압축
- `How it works`를 카드 3개 요약형으로 재정렬
- `Core features`를 2열 카드 묶음으로 재구성
- 하단 CTA를 단일 집중 카드로 정리
- 전체 섹션 간격을 다이어트해서 길이 대비 정보 밀도 개선

4. 비주얼 톤 개선
- 브랜드 컬러(#00bc7d, #8c52ff) 기반의 블러 글로우 레이어 확장
- 카드형 표면 계층(테두리/투명도/블러) 통일
- 라이트/다크 모두 동일한 레이아웃과 인터랙션 패턴 유지

5. 반응형 보정
- 모바일: 네브 요소 우측 액션 압축, 히어로 단일 컬럼
- 데스크톱: 히어로 2컬럼 + 정보 카드 분리
- 가로 스크롤 방지: 루트 `overflow-x-clip` 유지

6. 추가 보정: 플로팅 네브 브랜딩 정렬/크기
- 사용자 피드백 기준으로 브랜드 로고의 세로 중심 체감을 보정
  - `BrandLogo` 라벨에 `leading-none` + `inline-flex items-center` 적용
  - 링크 래퍼를 `inline-flex items-center`로 정렬 기준 통일
- 플로팅 네브 로고 크기를 소폭 확대
  - 마크: `h-9 w-9` (`sm` 이상 `h-10 w-10`)
  - 라벨 타이포: 한 단계 상향(`text-lg`~`md:text-[1.35rem]`)

7. 추가 보정: SaaS형 랜딩 -> Tool형 1스크린 랜딩 전환
- 사용자 피드백 기준으로 긴 설명 섹션/다중 스크롤 구조를 제거
- 데스크톱 기준 한 화면에서 핵심 행동(새 핸드 시작, 대시보드 진입)이 끝나는 구조로 재편
  - 상단 고정 플로팅 네브
  - 좌측: 짧은 카피 + 핵심 CTA
  - 우측: 즉시 이해 가능한 3-step 요약 카드
- 기존 하단 대형 footer/추가 섹션을 통합해 정보량을 압축
- 정책 링크는 데스크톱/모바일에서 각각 자연스럽게 접근 가능한 위치로 재배치
- 모바일은 콘텐츠 높이에 따라 자연 스크롤을 허용

8. 최종 보정: 컨셉 전환(비분할 + CTA 최소화)
- 랜딩을 좌우 분할형에서 단일 캔버스형으로 완전 전환
  - 중앙 1개 카드에서 메시지/행동/핵심 포인트를 모두 처리
- 상단바 액션을 요청대로 3개만 유지
  - 로그인 버튼
  - 언어 전환
  - 다크/라이트 토글
- CTA를 `새 핸드 시작` 1개 중심으로 정리해 클릭 선택 피로 제거
- 히어로 문구 강제 줄바꿈(`<br />`) 제거
  - 문장 단위 자연 줄바꿈으로 어색한 단어 분리 현상 완화

9. 다크모드 hover 이상 보정(ghost 계열)
- `Button`의 `ghost/sidebar/sidebarOutline`을 하드코딩 색상(slate 계열)에서 테마 토큰 기반으로 재정의
- 결과:
  - 다크모드에서 hover 시 배경/글자색 뒤집힘 체감 감소
  - 라이트/다크 모두 hover 톤 일관성 향상

10. 히어로 줄바꿈 보정(한국어 가독성)
- 사용자 피드백 기준으로 히어로 타이틀 줄바꿈을 재조정
  - `break-keep` 적용으로 한글 단어 내부 분리(예: `정 / 확하게`) 방지
  - 타이틀 폭을 `max-w-[26ch]`로 확장
  - `text-wrap: pretty` 적용으로 자연스러운 줄 나눔 유도

11. 핵심 포인트 문구 줄바꿈 보정(한 줄 유지)
- `featureItem` 영역을 3열 카드형에서 `inline chip` 레이아웃으로 변경
- 각 항목에 `whitespace-nowrap`를 적용해
  - `원하는 지점에서 Undo / Redo / Jump` 같은 문구가 불필요하게 2줄로 갈라지지 않게 수정

12. 플로팅바/본문 폭 동기화 보정
- 사용자 피드백 기준으로 상단 플로팅바 폭과 본문 카드 폭을 1:1로 맞춤
- 상단 `header`에 `mx-auto w-full max-w-4xl` 적용
  - 본문 섹션의 `max-w-4xl` 기준과 동일화

13. 대형 화면 줄바꿈 역전 현상 보정
- 사용자 피드백 기준으로 “화면이 커졌는데 오히려 히어로 문구가 줄바꿈되는” 현상 수정
- 조치:
  - 히어로 타이틀의 줄바꿈 정책을 `text-wrap: pretty` -> `text-wrap: wrap`으로 변경
  - 대형 화면에서 과도한 폰트 증가(`lg:text-6xl`) 제거
  - 내부 콘텐츠 최대 폭을 `max-w-3xl` -> `max-w-4xl`로 확장
